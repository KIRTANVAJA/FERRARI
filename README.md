# Ferrari Project

A modern and visually rich landing page inspired by Ferrari's brand and Formula 1 styling.

## Features

- Elegant Ferrari-themed design with a dark luxury aesthetic
- Responsive navigation bar with brand links and icons
- Hero section with a full-screen background image and bold headline
- Featured driver/player cards with hover effects
- Latest news section showcasing Formula 1 updates
- Sponsor and partner logo section
- Clean footer with brand information and copyright details

## Technologies Used

- HTML5
- CSS3
- Font Awesome icons
- Google Fonts

## Project Structure

- index.html - Main page structure
- style.css - Styling and layout
- accets/ - Images and assets used in the website

## How to Run

1. Open the project folder in your browser.
2. Launch index.html directly, or use a live server if available.

## Preview

This project is a static website designed to give a premium Ferrari-style presentation with strong visual storytelling and layout design.
